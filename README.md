# Port_scanner

## Network & Web Security Scanner (Lite-Nmap)
Высокопроизводительный асинхронный сканер для аудита сетевой инфраструктуры и анализа базовых уязвимостей веб-серверов. Разработан как pet-проект для изучения сетевых протоколов (L4/L7) и принципов многопоточной обработки данных.

## Ключевые возможности
Fast Port Scanning (L4): Многопоточный перебор портов с использованием ThreadPoolExecutor. Сканирование стандартного диапазона (1-1024) занимает менее 3 секунд.

Smart Target Discovery: Использование socket.getaddrinfo для корректного резолвинга доменов, поддержки IPv4/IPv6 и обработки сетевых ошибок.

Deep HTTP Analysis (L7): Автоматическое определение веб-сервисов на открытых портах (80/443).

Security Headers Audit: Проверка наличия критических заголовков безопасности:

Content-Security-Policy (защита от XSS)

X-Frame-Options (защита от Clickjacking)

X-Content-Type-Options (защита от MIME-sniffing)

Resilience: Обработка таймаутов и исключений (requests.exceptions, socket.gaierror), что гарантирует стабильность работы даже при нестабильном соединении.

## Технологический стек
Язык: Python 3.10+

Сетевое взаимодействие: socket (низкоуровневые сокеты), requests (HTTP-протокол).

Параллелизм: concurrent.futures (пул потоков).

## Установка и запуск
Клонируйте репозиторий:

Bash
git clone https://github.com/dmitriyryzhkov64-create/Port_scanner.git
cd Port_scanner
Создайте и активируйте виртуальное окружение:

Bash
python -m venv .venv
source .venv/bin/activate  # Для Windows: .venv\Scripts\activate
Установите зависимости:

Bash
pip install -r requirements.txt
Запустите сканер:

Bash
python main.py
